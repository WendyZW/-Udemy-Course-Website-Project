## Set-up instructions
 
0. open your command shell, for example Powershell in Windows 10,  change work dictionary to project folder.
1. create virtual enviroment using `python -m venv env`
2. Activate the virtualenv using command `source env/bin/activate` (Unix/macOS) or `.\env\Scripts\activate` (Windows)
3. Run command  `pip install -r requirements.txt` in your shell.
4. run the command `python .\main.py ` to run the app.
